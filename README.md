------------------------------------------------------------------------------------------------------
PROJET NGROK
------------------------------------------------------------------------------------------------------
Sur la base de ce repository (à forker) :

Créer une action GitHub qui va créer une image docker (via un Dockerfile) pour permettre de faire fonctionner votre serveur Flask.
Ce serveur Flask exposera le contenu de votre Repository GitHub sur Internet (en mode Preview) via la solution Ngrok qui vous fournira un DNS public temporaire. Le lien Internet temporaire sera accessible (lien visible depuis les logs de votre action GitHub) pendant 120 seconde.
